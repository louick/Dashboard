# ODS – Pará Dashboard (Dash)

Dashboard em Python/Dash para exploração dos indicadores ODS no Pará.
- UI clara, 3 colunas (6–6–5) para os 17 ODS
- Slider de **Ano de divulgação** (ano de referência **+2**)
- Séries exibidas **a partir de 2019** para evitar espaço morto
- Comparação cumulativa (Estado / Macro / Município) conforme o nível escolhido
